# Val's Day of Love & Kindness

A static, free-to-host kindness calendar website.

## How to use
Open `index.html` in a browser to preview.

## Free hosting
Upload these files to a GitHub repository, then turn on GitHub Pages:
Settings → Pages → Deploy from branch → main → root.

No ads, no logins, no database, no tracking.

Progress is saved locally in each person's browser using localStorage.
